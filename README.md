# AI_project

Steps to run:
<ul>
<li> cd to project location and activate virtual env using "venv\scripts\activate"  </li> 
<li> Install flask using pip with command "pip install flask flask-sqlalchemy" </li> 
<li> Run using "python run.py"   </li> 
</ul>
